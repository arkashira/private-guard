# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute Costs**: $0.10/user/month
- **Storage Costs**: $0.05/user/month
- **Bandwidth Costs**: $0.15/user/month  
**Total Cost per Active User**:  
**$0.10 + $0.05 + $0.15 = $0.30/user/month**

### Pricing Tiers
| Tier Name         | Price ($/mo) | Features                                           |
|-------------------|--------------|----------------------------------------------------|
| Basic             | $5           | Ad-free experience, basic privacy features         |
| Pro               | $10          | All Basic features + enhanced privacy controls     |
| Premium           | $15          | All Pro features + priority support + additional tools |

### Customer Acquisition Cost (CAC) Range
- **Estimated CAC**: $20 - $50 per user

### Lifetime Value (LTV) Estimate
- **Average Monthly Revenue per User (ARPU)**: 
  - Basic: $5
  - Pro: $10
  - Premium: $15
- **Average Customer Lifespan**: 24 months  
**LTV Calculation**:  
- Basic: $5 * 24 = $120  
- Pro: $10 * 24 = $240  
- Premium: $15 * 24 = $360  

### Break-even Users Count
**Break-even Point Calculation**:  
- Fixed Costs (assumed): $2,000/month  
- Contribution Margin per User:  
  - Basic: $5 - $0.30 = $4.70  
  - Pro: $10 - $0.30 = $9.70  
  - Premium: $15 - $0.30 = $14.70  

**Break-even Users Count**:  
- Basic: $2,000 / $4.70 ≈ 426 users  
- Pro: $2,000 / $9.70 ≈ 206 users  
- Premium: $2,000 / $14.70 ≈ 136 users  

### Path to $10K MRR
**Target MRR**: $10,000  
**User Count Required**:  
- Basic: $10,000 / $5 = 2,000 users  
- Pro: $10,000 / $10 = 1,000 users  
- Premium: $10,000 / $15 = 667 users  

**Recommended Path**:  
- Focus on **Pro Tier** with a target of **1,000 users** for a balanced offering and higher revenue potential.  
- Alternatively, target **Premium Tier** with **667 users** for maximum revenue per user.
